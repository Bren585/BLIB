# BLIB

This is my personal, custom game engine. 

It has a full rendering pipeline with deferred shading, shadows, and compatible with both GPU and CPU skinning. 

It also has a full physics engine, with collision handling in both 2D and 3D, for all kinds of collider shapes.

It has a simple particle system, a custom easy-use string system, text rendering, and built-in automatic threaded scene loading. 

It also uses a custom math library, built to integrate smoothly into DirectX11 and DirectXMath, with custom float2, float3, float4, quaternion, and transform classes.
